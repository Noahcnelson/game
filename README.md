# Chrono Serpent: Nexus Protocol

A high-intensity arcade survival game inspired by snake mechanics with layered systems:

- Physics-smoothed snake movement and segmented body simulation
- Enemy archetypes (runner, tank, sniper, boss)
- Active abilities (dash, temporal burst, mine deployment)
- Projectile combat and enemy bullet patterns
- Mission objectives per level with reward scoring
- Combo multiplier and level progression loop
- Shield + health resource model with particle effects and camera shake
- Responsive HUD and pause/restart controls

## Run

Open `index.html` in any modern browser.

## Controls

- **Move**: `WASD` or Arrow Keys
- **Shoot**: `Space`
- **Dash**: `Shift`
- **Temporal Burst**: `E`
- **Mine**: `Q`
- **Pause**: `P` or pause button
